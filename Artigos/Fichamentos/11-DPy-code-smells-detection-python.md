# Fichamento: DPy – Code Smells Detection Tool for Python

## Referência
Boloori, A., & Sharma, T. (s.d.). DPy: Code Smells Detection Tool for Python. Dalhousie University.

## Objetivo
Apresentar **DPy**, ferramenta de detecção de code smells para Python, cobrindo smells de design e de implementação e métricas de qualidade, com saída em CSV/JSON para mineração e estudos empíricos.

## Funcionalidades
- **Design smells**: oito tipos (quatro aplicáveis a classe e módulo).
- **Implementation smells**: onze tipos.
- Métricas de qualidade em nível de função/método, módulo e classe.
- Exportação CSV/JSON; pacote de replicação com dados de validação.

## Relação com os enunciados do laboratório
- **Lab 01**: RQ05 trata da **linguagem primária** dos repositórios; muitos dos 1000 mais populares podem ser em Python. DPy é **ferramenta de apoio** para laboratórios que incluam análise de qualidade de código em projetos Python (clone de repos, análise com DPy, correlação com estrelas/PRs/releases).
- **Medição e experimentação**: Ferramenta pronta para **pipelines de extração de métricas e smells** em estudos MSR; padrão de documentação e replicação (instruções, dados, scripts).
