# LabMedExp – Laboratório de Experimentação de Software

Repositório da disciplina **Laboratório de Experimentação de Software** (PUC Minas), com material de apoio, fichamentos de artigos e **entregas dos 5 laboratórios** (5 sprints).

Plano oficial da disciplina: [joaopauloaramuni/laboratorio-de-experimentacao-de-software → LABORATORIOS](https://github.com/joaopauloaramuni/laboratorio-de-experimentacao-de-software/tree/main/LABORATORIOS)

---

## Estrutura

| Pasta / arquivo | Conteúdo |
|-----------------|----------|
| **Artigos/** | PDFs de apoio e referência da disciplina |
| **Artigos/Fichamentos/** | Fichamentos em português com relação às RQs e medição/experimentação |

| **Auxiliar/** | Pasta apenas local — não é enviada ao GitHub (está no .gitignore) |

---

## Os 5 laboratórios

| Sprint | Tema |
|--------|------|
| **Lab 01** | Características de repositórios populares (GraphQL, 1000 repos, métricas) |
| **Lab 02** | Um estudo das características de qualidade de sistemas Java |
| **Lab 03** | Caracterizando a atividade de code review no GitHub |
| **Lab 04** | Visualização de dados utilizando uma ferramenta de BI |
| **Lab 05** | GraphQL vs REST – Um experimento controlado |

Detalhes e enunciados: **[Laboratorios/README.md](Laboratorios/README.md)** e na pasta de cada lab. Para o Lab-01, cada etapa (S01, S02, S03) possui subpasta própria com scripts, dados e relatórios independentes.

## Uso local

- **Lab 01:** usa um **Personal Access Token** do GitHub em variável de ambiente. Não versionar o token; ver `Laboratorios/Lab-01/QUERY-GRAPHQL-Lab01.md`.
